# nt-privacy-policy

Privacy policy for **지켜라! 냥냥수비대 / NyanTower: Cat Defense Squad** (Android · `com.catsori.nyan_tower`).

Hosted via GitHub Pages — see the live URL below for the URL to paste into Google Play Console.

## Live URLs

After enabling GitHub Pages on `main` branch (root):

- ko-KR: https://catsorigame.github.io/nt-privacy-policy/
- en-US: same URL, English tab

## Source

- [`index.html`](index.html) — bilingual (KO / EN) policy page used by Play Console
- [`privacy-ko.md`](privacy-ko.md) — Korean source
- [`privacy-en.md`](privacy-en.md) — English source

## Enabling GitHub Pages (one-time)

1. Push to `main`.
2. Repo → Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / `/ (root)`.
3. Wait ~1 min for build, then open the live URL.

## Updating

Edit `index.html` (and optionally `privacy-ko.md` / `privacy-en.md` for the Markdown source), bump the "Last updated" date, push to `main`. Pages redeploys automatically.
